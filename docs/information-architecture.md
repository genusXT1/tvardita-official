# Информационная архитектура (Information Architecture)

## 1. Главная страница (`/`)
- Сервисная панель (Языки, версия для слабовидящих, горячая линия, вход)
- Шапка (Логотип/Герб, Основное меню, Поиск, Кнопка "Обращение")
- Срочные уведомления (Alerts)
- Рабочая зона "Чем город может помочь вам сегодня?" (Поиск, Быстрые ссылки)
- Сегодня в городе (Часы работы, контакты, ближайшее заседание)
- Частые действия (Связаться, Расписание, Налоги, Документы)
- Актуальная информация (Объявления, Срочно, Публичные консультации)
- Новости города (Последние новости)
- Городской совет (Ближайшее заседание, решения)
- Открытая примэрия (Бюджет, закупки)
- Транспорт (Расписание)
- Городская жизнь (Культура, спорт, события)
- Контакты и подвал

## 2. Примэрия (`/city-hall`)
- Примар (`/city-hall/mayor`)
- Руководство (`/city-hall/leadership`)
- Аппарат примэрии (`/city-hall/departments`)
- Бюджет и финансы (`/city-hall/budget`)
- Закупки (`/city-hall/procurement`)
- Вакансии (`/city-hall/careers`)

## 3. Городской совет (`/council`)
- О совете (`/council/about`)
- Состав совета (`/council/members`)
- Заседания и повестки (`/council/meetings`)
- Проекты решений (`/council/draft-decisions`)
- Принятые решения (Реестр) (`/documents?type=decision`)

## 4. Жителям (`/services`)
- Муниципальные услуги (`/services/catalog`)
- Подать обращение (`/services/appeals/new`)
- Статус обращения (`/services/appeals/status`)
- Налоги и сборы (`/services/taxes`)
- Транспорт и расписание (`/services/transport`)
- Образование (`/services/education`)
- Культура и спорт (`/services/culture`)
- Муниципальные предприятия (Твардисан) (`/services/enterprises`)

## 5. Документы (`/documents`)
- Реестр всех официальных документов с фильтрами по типу, номеру, дате, статусу, языку.

## 6. Медиацентр
- Новости (`/news`)
- Объявления (`/announcements`)
- События (`/events`)

## 7. О городе (`/about`)
- История (`/about/history`)
- Галерея (`/about/gallery`)
- Почетные граждане (`/about/honorable-citizens`)

## 8. Контакты (`/contacts`)
- Телефоны, адреса, форма обратной связи.

## 9. Административная панель (`/admin`)
- Dashboard
- Управление контентом (Новости, Объявления, Документы)
- Управление городом (Сотрудники, Отделы, Предприятия, Транспорт)
- Обращения граждан (CRM для обращений)
- Системные настройки (Пользователи, Роли, Переводы, Журнал действий)
